## How to Contribute?
- Write code (see `DEVELOPMENT.org` in repository root)
- Playtest & report bugs
- Even just playing the game or supporting it is contributing in my book.

## How to Report Bugs?
1. Open an issue in the Issues tab
2. Do the same thing as No. 1, but with a discussion.
3. Contact me (see below)
Your bug report should include:
- What the bug is
- What OS you're on (e.g., Windows, Debian, Arch Linux btw)
- Preferrably how to reproduce the problem. You can skip this if the
  explanation of the bug already makes it clear
