This is a document explaining the controls of the game!
* [ and ] are not buttons that you have to press. It is just formatting.
  I doubt that anybody would be confused about this, but I make this note just in case.
  
Menu Navigation:
- Up: Up Arrow
- Down: Down Arrow
- Left: Left Arrow
- Right: Right Arrow
- Decrease Option: Left Arrow
- Increase Option: Right Arrow
- Select: Return / Enter
- Exit Menu: [X]

Gameplay (1D & 2D Modes):
- Move Left: A
- Move Right: D
- Pause: ESC

Gameplay (1D Mode):
- Jump: [W]
- Slow: [S]
- Stop Jump: [S]

Gameplay (2D Mode):
- Move Up: [W]
- Move Down: [S]
- Jump: Up Arrow
- Slow: Down Arrow
- Stop Jump: Down Arrow

Other:
- Take Screenshot: [F2]
